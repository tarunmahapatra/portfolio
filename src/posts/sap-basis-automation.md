# SAP BASIS Automation with Python and Selenium

As part of Dell's Finance ERP management team, I worked in the SAP BASIS group and built automation scripts that cut down repetitive manual work.

## What I built

- **Selenium and Python automation** for routine SAP-related tasks, reducing execution time by over 80%.
- **Reusable scripts** for UI-driven workflows that previously required manual clicks and checks.
- **Reliable execution flow** with error handling and logging so failures were easy to trace.

## Impact

The scripts replaced manual, repetitive work with trackable automation. The team spent less time on routine operations and more time on actual support issues, which made the platform more stable and predictable.

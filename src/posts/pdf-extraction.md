# PDF Data Extraction Tool

During my summer internship at Dell, I built a Python and OpenCV utility that extracts structured data from PDFs and presents it through a clean web interface.

## Highlights

- Used Python and OpenCV for image preprocessing and text region detection within PDF documents.
- Built a responsive HTML/CSS/JavaScript frontend for uploading PDFs and reviewing extracted results.
- Automated extraction of tabular and form-based content, reducing manual data-entry effort for the team.

## Tech stack

Python, OpenCV, HTML5, CSS3, JavaScript

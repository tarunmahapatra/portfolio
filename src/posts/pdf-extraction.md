# PDF Data Extraction Tool

During my summer internship at Dell, I built a Python + OpenCV utility that extracts structured data from PDFs and presents it through a clean web interface.

## Highlights

- Used Python and OpenCV for image preprocessing and text region detection.
- Built a responsive HTML/CSS/JavaScript frontend for uploading PDFs and reviewing extracted data.
- Reduced manual data-entry effort by automating the extraction of tabular and form content.

## Tech stack

Python, OpenCV, HTML5, CSS3, JavaScript

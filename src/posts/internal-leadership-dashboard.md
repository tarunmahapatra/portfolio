# Internal Dashboard for Leadership

I built a full-stack dashboard used by Dell's SAP BASIS leadership team to monitor operations and manage maintenance activities across more than 200 servers.

## What it does

- Pulls job-run logs into a single view using a Python backend and MongoDB storage.
- Lets operators trigger automation scripts directly from the UI during maintenance windows.
- Surfaces failure analytics so the team can spot recurring issues and prioritise fixes.

## Tech stack

React, Python, MongoDB

## Outcome

The dashboard turned scattered log data and manual status checks into one place for faster decision making. Maintenance activities became more reliable, and failure patterns were visible to leadership without ad-hoc reporting.
